传统色使用提示

怎么把颜色用起来

不要从概念开始。先确定颜色要完成的任务，再决定复制哪种色值或下载哪张色卡。

推荐流程

先做决定，再选颜色

每次只回答三个问题：用在哪里、占多大面积、最后交付到屏幕还是印刷。

01 找主色 在色卡库按色相或关键词筛出一张最接近气质的色卡。 02 看搭配 点击色卡图片，在弹窗里看背景、辅助、强调色怎么配。 03 再交付 界面复制 HEX/RGB，印刷或实物先下载色卡再做色差校验。

角色判断

先把颜色放到版面位置里看

背景 大面积铺底，优先低刺激、能承托文字的颜色。 内容 模块、图表和分区，优先选与主色气质接近的颜色。 重点 按钮、标题和数字，只用少量更有对比的颜色。
五色线索

青、赤、黄、白、黑，只用来帮你快速找方向

青清爽 / 生长 / 科技 赤提醒 / 节庆 / 热度 黄稳定 / 温润 / 中心 白留白 / 明净 / 呼吸 黑文字 / 边界 / 收束
复制格式

界面先用 HEX 或 RGB

网页、PPT、海报预览优先复制屏幕色值。要交给印厂时，再把色卡图和 CMYK 一起给对方复核。

交付检查

落地前看对比、光线和材质

同一个色值在屏幕、纸张、布料和釉面上会变样。重要项目不要只看屏幕预览。
